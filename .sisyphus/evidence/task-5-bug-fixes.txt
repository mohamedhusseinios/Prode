## Bug #22 Fix (lines 32-34): Validate config before save

**Change**: Added type check in save_config() before writing.

**Before**:
```python
def save_config(config: ProviderConfig) -> None:
    """Persist provider config to disk."""
    CONFIG_PATH.write_text(json.dumps(asdict(config), indent=2))
```

**After**:
```python
def save_config(config: ProviderConfig) -> None:
    """Persist provider config to disk."""
    if not isinstance(config, ProviderConfig):
        raise TypeError("config must be ProviderConfig")
    CONFIG_PATH.write_text(json.dumps(asdict(config), indent=2))
```
